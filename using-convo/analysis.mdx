After your sessions complete, Convo analyzes the transcripts and recordings to generate a full set of results. This page covers how to trigger analysis and what gets produced.

## Triggering analysis

You can trigger analysis from the study dashboard once sessions have completed. Click **Run Analysis** to start the process.

<Frame>
  [IMAGE: The study dashboard showing the Run Analysis button and session completion status]
</Frame>

<Info>
  You can run analysis after each session completes, or wait until all sessions are done and analyze everything at once. Running analysis on all sessions together produces the best cross-session insights.
</Info>

---

## Pipeline stages

Analysis moves through five stages:

| Stage | What happens |
|-------|-------------|
| **Pending** | Analysis is queued and waiting to start |
| **Preparing** | Transcripts and recordings are being processed |
| **Analyzing** | AI is identifying themes, pulling quotes, and generating insights |
| **Generating clips** | Video clips are being created from session recordings |
| **Complete** | All results are ready to view |

<Frame>
  [IMAGE: The analysis progress indicator showing the pipeline stages]
</Frame>

Analysis typically takes a few minutes, depending on the number of sessions and total discussion time. You will be notified when results are ready.

## What gets generated

Convo produces six types of results from your sessions:

<Columns cols={2}>
  <Card title="Summary" icon="file-lines" href="/results/summary">
    High-level findings, decisions, agreements, and debates. Includes metrics like consensus rate and participant count.
  </Card>
  <Card title="Deep Dive and Responses" icon="list-check" href="/results/deep-dive-and-responses">
    Moment-by-moment analysis and per-participant response views.
  </Card>
  <Card title="Themes" icon="layer-group" href="/results/themes">
    AI-identified patterns across all sessions with supporting quotes and participant attribution.
  </Card>
  <Card title="Video Clips" icon="film" href="/results/video-clips">
    Auto-generated video clips of key moments from your sessions.
  </Card>
  <Card title="Explore" icon="comments" href="/results/explore">
    An AI chat interface to ask questions about your study data and generate charts.
  </Card>
</Columns>

Each result type serves a different purpose. The **Summary** gives you the big picture. **Deep Dive** and **Responses** let you go moment-by-moment. **Themes** show patterns across participants. **Video Clips** give you shareable video evidence. **Explore** lets you ask follow-up questions on demand.
